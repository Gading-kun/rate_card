# Rate Card

A clean, responsive rate card web application for a creative studio. Displays service packages, hourly rates, and add-ons with a built-in inquiry form.

## Features

- **Three service categories** — Graphic Design, Web Development, Photography — each with Starter / Pro / Enterprise tiers
- **Hourly rates table** for UI/UX, Front-End, Back-End, Copywriting, and Video Editing
- **Add-ons & Extras** section
- **Currency toggle** — switch prices between USD and IDR instantly
- **Contact / inquiry form** with client-side validation
- **Print / Save PDF** button with print-optimized styles
- **FAQ** accordion section
- Fully responsive (mobile-first) with no external dependencies

## Usage

Open `index.html` in any modern browser — no build step required.

```
rate_card/
├── index.html   # Main page
├── styles.css   # All styles
└── script.js    # Currency toggle, form validation, print
```
